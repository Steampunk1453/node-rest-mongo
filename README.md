# node-rest-mongo

## Clone this repository
```
git clone https://github.com/Steampunk1453/node-rest-mongo.git
```

## Then install the dependencies
```
npm install
```

## MongoDB
```
brew install mongodb
```

```
sudo chown -R `id -un` /data/db
```

Run the Mongo daemon
```
 mongod
```

## Start the server

Run in development mode
```
npm run dev
```
